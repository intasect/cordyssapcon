INSTALATION DETAILS
===================

In order for the SAP connector to work you need to copy the JCo binaries and DLLs first.
Copy the following files to the <cordys_install_dir>/BAC/SAP/lib folder:
	- jdsr.jar
	- sapidoc.jar
	- sapidocjco.jar
	- sapjco.jar
	- librfc32.dll
	- msvcp71.dll
	- msvcr71.dll
	- sapjcorfc.dll
